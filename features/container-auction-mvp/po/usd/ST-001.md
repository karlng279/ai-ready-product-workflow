# ST-001: Acceptance Criteria

**Story:** Seller lists container for sale
**USL Reference:** [ST-001 in usl.md](../usl.md#st-001-seller-lists-container-for-sale)
**Last Updated:** 2026-02-03

---

## UI Elements

- **AC-001:** The "Create Listing" page must have a form with the following fields: Title, Description, Quantity, Price, Location, Type (e.g., 20ft, 40ft), Condition (New, Used, Damaged).
- **AC-002:** The form must include an image upload section supporting multiple images (JPG, PNG).
- **AC-003:** A "Publish Listing" button must be present at the bottom of the form.
- **AC-004:** A "Save as Draft" button must be present.

---

## UI Behavior

- **AC-005:** Clicking "Publish Listing" validates all required fields.
- **AC-006:** If validation fails, error messages are displayed inline below the invalid fields.
- **AC-007:** Upon successful publication, the user is redirected to the "My Listings" page with a success toast notification.
- **AC-008:** Uploaded images must show a preview thumbnail with a "Remove" option.

---

## Logic

- **AC-009:** Price must be a positive numeric value.
- **AC-010:** Title must be at least 10 characters long.
- **AC-011:** At least one image is required for publication.

---

## Special Notes

- **AC-012:** Supported localized currencies: VND, USD.
- **AC-013:** Mobile responsive: Form should stack vertically on small screens.

---

## Non-Functional Requirements

- **NFR-001:** Image upload must complete within 5 seconds for files < 5MB.
- **NFR-002:** Form submission must process within 2 seconds.
