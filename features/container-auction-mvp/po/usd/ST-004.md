# ST-004: Acceptance Criteria

**Story:** Buyer browses and searches containers
**USL Reference:** [ST-004 in usl.md](../usl.md#st-004-buyer-browses-and-searches-containers)
**Last Updated:** 2026-02-03

---

## UI Elements

- **AC-001:** Marketplace Homepage features a prominently displayed Search Bar.
- **AC-002:** Filters sidebar/panel includes: Container Type, Condition, Price Range, Location, Listing Type (Sale vs Auction).
- **AC-003:** Results grid displaying container cards with thumbnail, title, price, and location.

---

## UI Behavior

- **AC-004:** Search updates results in real-time or upon pressing "Enter".
- **AC-005:** Changing filters automatically refreshes the results grid.
- **AC-006:** Clicking a card navigates to the Container Detail Page.

---

## Logic

- **AC-007:** Default sort order is "Newest Listed".
- **AC-008:** Search matches against Title, Description, and Location fields.
- **AC-009:** Expired auctions or sold items are excluded from default search (unless "Show sold" is checked).

---

## Special Notes

- **AC-010:** Empty state design required when no results match.

---

## Non-Functional Requirements

- **NFR-001:** Search results filter execution < 500ms.
