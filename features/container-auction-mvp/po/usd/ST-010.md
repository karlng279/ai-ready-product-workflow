## UI Elements

- **AC-001:** "My Request" detail view shows list of Received Quotes.
- **AC-002:** Each quote shows: Provider Name, Cost, Duration, Comments.
- **AC-003:** "Accept Quote" button next to each option.

---

## UI Behavior

- **AC-004:** Clicking "Accept Quote" triggers confirmation modal.
- **AC-005:** Post-acceptance, request status changes to "In Progress".
- **AC-006:** Provider contact info is revealed to the Owner.

---

## Logic

- **AC-007:** Accepting a quote rejects all other pending quotes for this job.

---

## Special Notes

- **AC-008:** Email triggers to both parties to connect them.
