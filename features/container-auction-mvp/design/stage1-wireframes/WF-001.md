# Wireframe: Create Listing (Sale)

**Wireframe ID:** WF-001
**Feature:** Container Auction MVP
**Story:** ST-001 - Seller lists container for sale
**Addresses:** AC-001, AC-002, AC-003, AC-004, AC-005, AC-006, AC-008, AC-009, AC-010, AC-011
**Screen Type:** Form
**Theme:** MDS (Default)
**Theme File:** ../../../design-framework/themes/mds.md

---

## Theme Context

**Selected Theme:** MDS

**Theme Impact on Design:**
- **Visual Style:** Modern & bold, cleaner layout, high contrast
- **Color Approach:** White/Gray background, Magenta primary actions, Teal accents
- **Pattern References:** Standard form pattern with clear validation states
- **Component Defaults:** Large click targets, distinct primary buttons

---

## Purpose

Allows sellers (Carriers/SMEs) to create a new fixed-price listing for a container.

---

## Layout Structure

### Desktop (> 1024px)

**Page Structure:**
- Header: Fixed top nav
- Main Content: Centered single-column form (max-width 800px)
- Footer: Standard footer

### Mobile (< 640px)

**Layout Changes:**
- Full width form
- Sticky action bar at bottom maybe (or just standard stack)

---

## UI Elements Detail

### Main Content

**Section 1: Header**
- Title: "Create New Listing" (h1)
- Breadcrumbs: Home > Inventory > New Listing

**Section 2: Basic Info (Card)**
- Title Input: Wide text field
- Type Selector: Dropdown (20ft, 40ft, etc.)
- Condition Selector: Chips/Radio (New, Used, Damaged)
- Location Input: Autocomplete field

**Section 3: Pricing & Media (Card)**
- Price Input: Number with currency selector (VND/USD)
- Quantity Input: Number
- Description: Textarea
- Image Upload: Drag & drop zone + Grid of previews

**Section 4: Actions**
- "Save as Draft" (Secondary/Outline)
- "Publish Listing" (Primary/Magenta)

---

## Interaction / States

- **Validation:** Live validation on blur (AC-006)
- **Image Upload:** Immediate preview with delete overlay (AC-008)
- **Submit:** Button shows loading spinner, then redirects to My Listings

---

## Components Required

- ShadCN Form (Input, Select, Textarea, Hook Form)
- ShadCN Button (msg-primary, outline)
- ShadCN Card
- Custom: ImageUploader
