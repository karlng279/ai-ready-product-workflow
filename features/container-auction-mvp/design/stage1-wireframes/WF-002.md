# Wireframe: Create Auction

**Wireframe ID:** WF-002
**Feature:** Container Auction MVP
**Story:** ST-002 - Seller creates container auction
**Addresses:** AC-001, AC-002, AC-003, AC-004, AC-010
**Screen Type:** Form
**Theme:** MDS (Default)

---

## Purpose

Allows sellers to create an auction-style listing.

---

## Layout Structure

Similar to WF-001, but with auction-specific fields in the Pricing card.

---

## UI Elements Detail

### Main Content Differences from WF-001

**Section: Auction Settings**
- Starting Bid (instead of Price)
- Reserve Price (Optional)
- Duration: Toggle/Select (3d, 7d, 14d) containing calculated End Date text.
- Buy Now Price (Optional - future scope, but good to have in UI structure)

**Action:** "Start Auction" (Primary) - shows warning modal (AC-006)
