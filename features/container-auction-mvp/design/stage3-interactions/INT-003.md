# Interaction Flow: Repair Workflow

**Interaction ID:** INT-003
**Interaction Name:** Repair Workflow (Request -> Quote -> Accept)
**Component ID(s):** COMP-007, COMP-009, COMP-010
**Interaction Type:** Multi-User Workflow
**Theme Reference:** mds

---

## ui-ux-pro-max Prompt

Implement Repair Workflow using MDS theme:

**Pattern:** Request-Response Workflow

**States (Lifecycle):**
- **Draft Request:** Owner creating.
- **Open:** Published, waiting for quotes.
- **Quoted:** Provider submitting quote.
- **Review:** Owner reviewing quotes.
- **Assigned:** Owner accepted a quote.
- **In Progress:** Repair started.
- **Completed:** Repair finished.

**Transitions:**
- Draft -> Open: Owner submits COMP-007.
- Open -> Quoted: Provider submits COMP-009.
- Quoted -> Assigned: Owner accepts in COMP-010.
